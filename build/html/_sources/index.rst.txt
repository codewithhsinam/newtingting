TingTing documentation
======================

TingTing is an AI-powered telephony system that lets you send programmed voice calls in Nepali to thousands of
recipients with just a few clicks. Features include scheduling tools and customized messaging options. 
The TingTing API is well-documented and easy to use, providing maximum flexibility and customization for developers 
building custom applications.
